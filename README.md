# 前端轮播开发方法

### 引入Gulp打包自动构建

 ```
 安装
 npm install
 
 启动
 npm start
 
 ```
 #### 好处
 
 1. git快速下载包，保证每次都是最新，最完美新构建架包
 2. 引入sass加快开发效率，自动压缩css
 3. 自动压缩图片
 4. npm引入，急速包引入
 5. 页面同步刷新(永远不需要再不停保存与刷新)
